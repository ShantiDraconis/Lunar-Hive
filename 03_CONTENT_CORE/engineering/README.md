## Core Principles

1. **Constraints define the problem** (not obstacles)
2. **Prototype over perfection** (iterate based on reality)
3. **Safety is not optional** (analyze and mitigate risks)
4. **Integrate sustainability** (resource, environment, ethics)

See `/00_MANIFEST/PRINCIPLES.md` and `/docs/axes/ENGINEERING.md`
