## Number System Possibilities

### Base Systems

**Base 10 (Decimal):**
- Most common human counting base
- Evidence: Harappan weight system shows decimal component
- Hypothesis: Script may use decimal numerals
- Confidence: Low-Medium

**Base 8 (Octal) / Base 16 (Hexadecimal):**
- Evidence: Harappan weights show binary progression
- Hypothesis: Mixed base system possible
- Confidence: Low

**Base 20 (Vigesimal):**
- Less common, but attested in some cultures
- No strong evidence in Harappan context
- Confidence: Very low

**Mixed Base Systems:**
- Combination of bases (e.g., 12 for time, 10 for counting)
- Many ancient cultures used mixed systems
- Cannot rule out
- Confidence: Speculative

### Additive vs. Positional

**Additive System:**
- Symbol repetition indicates addition (e.g., III = 3)
- Simpler system, common in early writing
- Stroke repetitions suggest this is possible
- Confidence: Low-Medium

**Positional System:**
- Value depends on position (like modern decimal)
- More sophisticated
- Less evidence for this in Harappan
- Confidence: Low

**Hybrid System:**
- Combination of both approaches
- Common in many ancient systems (Roman numerals, Egyptian)
- Possible but unproven
- Confidence: Speculative

## Comparative Numeric Systems

### Proto-Cuneiform Numerals
- Well-understood numeric system
- Multiple bases depending on commodity counted
- Separate numeric symbols distinct from words

**Relevance:** Model of how early numeric systems work
**Caution:** Different system, different culture

### Egyptian Numerals
- Decimal system with distinct symbols for powers of 10
- Additive system
- Clear numeric symbols

**Relevance:** Contemporary numeric thinking
**Caution:** Different script tradition

### Roman Numerals
- Mix of additive and subtractive principles
- Limited symbol set (I, V, X, L, C, D, M)
- Some symbols have multiple functions (C = 100, C = first letter of "Centum")

**Relevance:** Shows complexity possible in "simple" systems
**Caution:** Much later development

### Chinese-Japanese Numerals
- Characters with numeric and semantic values
- Multiple reading systems
- Context-dependent interpretation

**Relevance:** Shows numeric-semantic ambiguity in real scripts
**Caution:** Entirely different linguistic context

## Falsification Criteria

### Would Strengthen Numeric Hypothesis:
- Discovery of clearly numeric context (abacus, counting board)
- Symbols appearing in systematic progressions
- Statistical pattern matching known numeric distributions
- Archaeological context indicating accounting

### Would Weaken Numeric Hypothesis:
- Random distribution of suspected numerals
- No pattern in suspected number sequences
- Better alternative interpretation emerges
- Contextual evidence contradicts numeric use

### Would Falsify Numeric Hypothesis:
- Definitive decipherment showing non-numeric meanings
- Proven usage incompatible with numeric function
- Archaeological context excluding numeric interpretation
