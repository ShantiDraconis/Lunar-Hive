# Code of Conduct

## Purpose

The Lunar Hive is a **public archive** and **conceptual foundation**. It exists for rigorous observation, documentation, and discussion of ideas.

This is not a social space. This is a working space.

## Standards

### Expected Behavior

- Focus on ideas, not individuals
- Provide evidence and citations when making claims
- Respect silence as intentional
- Keep contributions concise and structured
- Acknowledge uncertainty clearly

### Unacceptable Behavior

- Personal attacks or harassment
- Deliberate misinformation
- Commercial promotion or spam
- Off-topic content
- Emotional manipulation
- Sharing private information without consent

## Scope

This Code of Conduct applies to:
- All repository spaces (issues, discussions, pull requests)
- Public references to this project
- Representation of the project in external venues

## Enforcement

Violations will be addressed through:
1. Warning with explanation
2. Temporary restriction of access
3. Permanent removal from participation

Enforcement decisions are made by repository maintainers and are final.

## Reporting

To report violations, open an issue labeled "conduct" or contact maintainers directly through GitHub.

Reports will be handled with discretion.

## Attribution

This Code of Conduct is designed specifically for the nature and purpose of the Lunar Hive project.
